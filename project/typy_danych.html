Typy Danych w Pythonie

Python obsługuje różnorodne typy danych, które pozwalają na przechowywanie różnych rodzajów informacji.

Podstawowe typy danych

  • int: liczby całkowite, np. 5, -10, 100
  • float: liczby zmiennoprzecinkowe, np. 3.14, -0.5, 2.0
  • str: ciągi znaków, np. "Hello, World!", 'Python'
  • bool: wartości logiczne, True lub False

Operacje na typach danych

Python automatycznie przypisuje typy zmiennych na podstawie przypisanej wartości. Możesz również konwertować typy danych, np. z int na float lub str na int, za pomocą funkcji takich jak int(), float(), str().

x = 5        # int
y = 3.14     # float
z = "Hello"  # str

# Konwersja typów
a = float(x)  # Konwertowanie int na float
b = int(y)    # Konwertowanie float na int
c = str(x)    # Konwertowanie int na str
