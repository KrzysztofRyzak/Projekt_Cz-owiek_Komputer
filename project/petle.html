Pętle w Pythonie

Pętle w Pythonie umożliwiają wykonywanie bloków kodu wielokrotnie, dopóki spełniony jest warunek.

Rodzaje pętli

  • for
  • while

Przykłady użycia pętli

# Pętla for
for i in range(5):
    print(i)

# Pętla while
x = 0
while x < 5:
    print(x)
    x += 1
